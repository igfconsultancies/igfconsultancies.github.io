Requirements

OS X 10.7

Xcode 4.3.2+

iOS 5

What you will learn

  • How to use manage version control GIT repository
  • How to geolocate entities
  • How to create a model
  • How to import open source

SOP

  1. Create a new Xcode project GeoPhotos Storyboard ARC and git options
  2. Drag out a Split View Controller from the inspector objects
  3. Add a new file NSObject (PhotoManager model)
  4. Fetch list of photos (must be geotagged with entity search term) ideally with an asychronous block call Keep the list in memory (cache?) Make the data publically accessible Cache the photo if memory is sufficient. If not, purge cache (think about adding convenience methods to call classes) Might need to define relevant global variables such as latitude and longitude Create singlet + (id)sharedManager;
  5. Drag in Stanford flickr FlickrAPIKey (apply for flickr key) and FlickrFetcher.h/m
  6. Add CoreLocation, Mapkit and CoreGraphics frameworks
  7. Modify the iPad storyboard
  8. Should compile without errors
  9. Add Entity Annotations and Photo classes
  10. Add new file CalloutViewController
  11. Add HJCache
  12. Add MKMapView+ZoomLevel
  13. Wire up the iPhone storyboard
